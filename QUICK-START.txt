=== Nexus电商系统 - 快速启动 ===

如果批处理脚本无法运行，请尝试以下方法：

方法1：使用批处理脚本（推荐）
--------------------------------
1. 双击 "start-clean.bat" 启动服务
2. 双击 "stop-clean.bat" 停止服务

或从命令行运行：
.\start-clean.bat

方法2：手动启动
---------------
1. 启动后端：
   cd nexus-backend
   mvn spring-boot:run

   （新窗口）

2. 启动前端：
   cd nexus-frontend
   npm run dev

停止服务：
---------
使用对应的方法停止服务：
- 批处理: stop-clean.bat
- 手动: 在两个命令行窗口按 Ctrl+C

常见问题：
---------
Q: 脚本双击没有反应？
A: 使用PowerShell脚本或手动启动

详细说明请查看 README.md 和 manual-start.md
